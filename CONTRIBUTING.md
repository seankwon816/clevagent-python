# Contributing to ClevAgent SDK

Thanks for your interest in contributing!

## Getting Started

1. Fork this repo
2. Clone your fork: `git clone https://github.com/YOUR_USERNAME/clevagent-python.git`
3. Install in dev mode: `pip install -e ".[all]"`
4. Make your changes
5. Submit a PR

## Guidelines

- Keep it simple — this SDK is intentionally minimal (1 dependency: `requests`)
- Add type hints to function signatures
- Test your changes manually against https://clevagent.io or a local backend

## Reporting Issues

Open an issue at https://github.com/seankwon816/clevagent-python/issues
